# Media Strategy for LinkedIn

The media (image, video, carousel, or PDF) is the most important visual element of a LinkedIn post. It takes up roughly 80% of the screen. The hook takes 20%. The rest takes 0%. Treat media like a book cover - it's the first thing people evaluate.

## Core Principles

1. **Scrappy > polished.** The more something looks like an ad, the worse it performs. Organic-looking media outperforms branded, designed content.
2. **Useful and saveable.** The best media makes someone want to screenshot it, bookmark it, or DM it to a colleague.
3. **1080 x 1350 is the format.** Single vertical image, optimized for mobile feeds.

## Common Misconceptions

- "The text is what matters" - No. Only the hook and media matter. A wall of text without compelling media gets scrolled past.
- "Videos are the best format" - No. People are supposed to be working on LinkedIn, not binge-watching. Video has its place but isn't the default winner.
- "I need strong branding on my media" - No. Heavy branding makes it look like an ad. Be scrappy.

## What Performs Best

### Tier 1: Single Image (Default Recommendation)
- 1080 x 1350 pixels
- Looks handmade or scrappy, not corporate
- Contains useful information (cheat sheet, framework, data visualization, comparison)
- Makes someone want to save it or send it to a colleague

### Tier 2: PDF Carousel
- For content with 5+ distinct points that benefit from visual separation
- Hook slide must stop the scroll on its own
- 7-9 slides of real value
- Clean design, no heavy branding
- Final slide: clear CTA
- Hand off to carousel-skill for execution

### Tier 3: Text-Only
- For posts where the story or hot take is strong enough to stand alone
- Best for Format 1 (personal story) and Format 3 (hot take) when the narrative is compelling
- The hook does all the heavy lifting here

### Tier 4: Video
- Keep under 90 seconds
- Hook in first 3 seconds
- Add captions (most people watch muted on LinkedIn)
- Best for announcements, behind-the-scenes, or when personality/energy is the point

## Creating Visual Media with AI

When Hannah needs a visual (cheat sheet, infographic, data visualization), here's the recommended workflow:

### For Infographic-Style Images
1. **Find inspiration on Pinterest.** Search for the niche topic + "graph," "cheat sheet," or "infographic."
2. **Extract the structure.** Use an AI tool to reverse-engineer the layout and information hierarchy from a reference image.
3. **Find a style reference.** Search Pinterest for "handwritten style infographic" or organic/scrappy visual styles. The goal is to NOT look like an ad.
4. **Generate the visual.** Combine the content with the scrappy style reference using an image generation AI.

### Style Guidelines for AI-Generated Visuals
- Prefer handwritten, sketch-like, or organic styles over clean corporate design
- Muted or warm color palettes over bright neon or brand colors
- Readable text at mobile size (test it)
- No stock photo vibes
- No watermarks or obvious AI artifacts

## Media Recommendations by Post Format

| Format | Default Media | Alternative |
|---|---|---|
| Personal Story (F1) | Text-only or candid photo | Single image with key quote |
| Numbered List (F2) | Single image cheat sheet | PDF carousel |
| Hot Take (F3) | Text-only | Single image with bold statement |
| Resource/Template (F4) | Single image infographic | PDF carousel for complex frameworks |
| Someone's Story (F5) | Photo of the person (if available) | Text-only |

## When Suggesting Media

Always include a media recommendation when presenting a post draft. Be specific:

- **If single image:** Describe what it should contain and the visual style
- **If carousel:** Note that you'll hand off to carousel-skill, or provide the slide-by-slide content brief
- **If text-only:** Say so explicitly - it's a valid choice, not a fallback
- **If applicable:** Suggest a Pinterest search query for style inspiration
